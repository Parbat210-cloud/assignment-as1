Book Ordering System (Python)

This "Book Ordering System" interface allows users to add customers and books, place orders, calculate shipping, and generate invoices. It includes dropdowns to select customers and books, urgent shipping option, and buttons to search or view all invoices.

Sum of Digit Calculator (Python)

This Python program calculates the sum of single-digit numbers entered by the user. It takes input as a string, iterates through each character, converts it to an integer, and adds it to a running total. Finally, it prints the total sum. The code demonstrates input handling, loops, type conversion, and basic arithmetic operations in a simple, beginner-friendly project.
